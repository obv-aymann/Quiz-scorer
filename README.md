Quiz Scorer:
Created by: Ayman
Project Description:
This is a Python Quiz Scorer program that calculates:
- Total score
- Average score
- Highest score
- Lowest score
- Add new score

The program can:
- Calculate total score
- Calculate average score
- Find highest score
- Find lowest score
- Add a new score to the list

Functions Included:
`calculate_total(scores)`
`calculate_average(scores)`
`get_highest(scores)`
`get_lowest(scores)`
`add_score(scores, new_score)`

The tests verify:
- Total calculation
- Average calculation
- Highest score
- Lowest score
- Adding new scores
